# dotfiles

My personal development environment and system configuration files.

This setup is based on [`nelsonmestevao/dotfiles`](https://github.com/nelsonmestevao/dotfiles), adapted to my own workflow.

## What’s inside

- Shell configuration
- Nix-based environment/configuration
- Editor tooling and UI tweaks

## Usage

```bash
git clone https://github.com/0xDracula/dotfiles.git
cd dotfiles
```

Then follow the main repo for instructions regarding installation

## Notes

- Keep secrets and machine-specific values out of version control.
- Make small, tested changes to keep the setup stable.
